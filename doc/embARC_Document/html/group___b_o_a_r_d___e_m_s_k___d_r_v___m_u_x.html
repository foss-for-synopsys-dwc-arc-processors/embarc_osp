Detailed Description

EMSK Mux Controller Driver.

Mux controller is the hardware external PMOD port pin connection controller, it can distribute the external port pins into different functions like general input/output, spi, iic, uart and so on.

#define SPI_MAP_CTRL   2

32-bits, offset 0x8, SPI_MAP_CTRL[0] selects the mode of operation of the SPI Slave:
