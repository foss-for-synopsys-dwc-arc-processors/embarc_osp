Detailed Description

definitions for defining command code

here defines macros to define command code, in system code, use DEV_SET_SYSCMD to define command code. in user code, use DEV_SET_USRCMD to define command code. So that there will be no conflicts in system and user defined command code. this used used in UART Device HAL Interface, SPI Device HAL Interface, IIC Device HAL Interface, GPIO Device HAL Interface, and in user code
