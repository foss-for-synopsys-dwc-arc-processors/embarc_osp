Detailed Description

Macros for IIC device related configurations.

During IIC device during implementation, it is recommended that when user set iic speed to a speed maybe not supported in your iic device, device driver will set to a supported slower speed.

IIC_SPEED_HIGH

IIC_SPEED_ULTRA 

Unidirectional(Write only), Ultra Fast-mode (UFm), with a bit rate up to 5 Mbit/s
