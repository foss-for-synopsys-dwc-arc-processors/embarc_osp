Detailed Description

embARC IoT Amazon Web Services(AWS) Smart Home Node Front Door

Extra Required Tools

Extra Required Peripherals

  • Digilent PMOD WIFI(MRF24WG0MA)
  • Digilent PMOD TMP2

Design Concept

The Pmod modules should be connected to EMSK. This example is designed to show how to connect EMSK and AWS IoT with using embARC.

Usage Manual

Before compiling this example, you need to change macro AWS_IOT_MQTT_HOST in aws_iot_config.h to your own aws iot cloud mqtt host. And create an aws iot thing named frontdoor, you can refer to Gettting Started with AWS IoT for details, and generate certificates and download them and rename them to the ones under aws/smarthome_nodes/cert/frontdoor.

Copy aws/smarthome_nodes/cert to root folder of the EMSK SD card and run this example. It can report the following parameters to the AWS IoT Shadow.

  • the FrontDoor Lock (locked or unlocked) - Button X <-> LED 4-5

EMSK can send the status of the FrontDoor Lock status and temperatue to the AWS IoT and interact with AWS IoT with using embARC. The multiple nodes of smart home can connect AWS IoT cloud simultaneously.

Open dashboard website, and load dashboard-smarthome-multithings.json dashboard configuration in upper folder, and then you can control and monitor the multiple nodes.

After load the configuration, you also need to click at the setting icon, and then click on the aws datasource, and then change the AWS IOT ENDPOINT, REGION, ACCESS KEY, SECRET KEY to your own aws ones.
