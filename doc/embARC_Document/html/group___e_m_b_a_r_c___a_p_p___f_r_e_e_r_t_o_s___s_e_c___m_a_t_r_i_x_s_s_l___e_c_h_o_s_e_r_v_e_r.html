Detailed Description

embARC Example for secure echoserver based on FreeRTOS+LwIP and MatrixSSL

Extra Required Tools

Extra Required Peripherals

  • Digilent PMOD WIFI(MRF24WG0MA)

Design Concept

This example is designed as a secure echoserver. The communication is based on TLS. This example needs to work with echoclient example of MatrixSSL in pair.

Usage Manual

  • Hardware:
    • EMSK node1: PMOD WiFi to J5 EMSK
    • EMSK node2: PMOD WiFi to J5 EMSK
  • WiFi Network: SSID:embARC, PASSWORD:qazwsxedc
  • Run steps:
    • This example should work together with the EMBARC_APP_FREERTOS_SEC_MATRIXSSL_FREERTOS_ECHOCLIENT.
    • This example(echoserver) should run first, get the IP.
    • change the IP of echoserver in echoclient source code, build and run echoclient.
    • the exchange of information will be shown in terminal as below.
ScreenShot of MatrixSSL secure echoserver
